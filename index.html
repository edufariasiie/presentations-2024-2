Introdução

A manipulação de strings e números no Python é fundamental para programadores.

Strings são sequências de caracteres que podem ser modificadas e analisadas.

Números permitem operações matemáticas robustas e precisas.

Objetivo: Entender as operações manipulação em Python.

Operações Básicas com Strings

Concatenação: Uso do operador +.


nome_completo = "Eduardo" + " Faria"
print(nome_completo)  # Eduardo Faria

Repetição: Uso do operador *.


repeticao = "Python " * 3
print(repeticao)  # Python Python Python

Indexação e Fatiamento:


palavra = "Programação"
print(palavra[0])  # P
print(palavra[-1])  # o
print(palavra[0:5])  # Progr

Métodos de Manipulação de Strings

Alterar o caso das letras:


texto = "Python é ótimo!"
print(texto.upper())  # PYTHON É ÓTIMO!
print(texto.lower())  # python é ótimo!

Remover espaços em branco:


nome = " Eduardo "
print(nome.strip())  # Eduardo

Substituir partes da string:


saudacao = "Olá, Mundo!"
print(saudacao.replace("Mundo", "Python"))  # Olá, Python!

Escapando Caracteres em Strings

Sequências de escape: Para incluir caracteres especiais.

  • Nova linha: \n
  • Tabulação: \t

texto = "Linha 1\nLinha 2\tcom tabulação"
print(texto)
# Saída:
# Linha 1
# Linha 2    com tabulação

Operações Aritméticas Básicas com Números

  • Adição: +
  • Subtração: -
  • Multiplicação: *
  • Divisão: /
  • Exponenciação: **

x = 10
y = 3
print(x + y)  # 13
print(x / y)  # 3.3333
print(x ** y)  # 1000

Precisão e Tipos de Dados Numéricos

Inteiros (int): Para contagens exatas.

Ponto flutuante (float): Para cálculos que requerem frações decimais.

Conversão entre tipos:


numero_str = "123"
numero_int = int(numero_str)
numero_float = float(numero_str)
print(numero_int, numero_float)  # 123 123.0

Considerações Finais

A manipulação de strings e números é uma habilidade essencial no Python.

Dominar essas técnicas permite processar dados de forma eficiente em diversas áreas.

O Python oferece uma ampla gama de métodos e operações para facilitar o trabalho com esses tipos de dados.
